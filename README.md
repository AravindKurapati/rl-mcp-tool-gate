# rl-mcp-tool-gate

RL-trained pre-flight gate that prunes MCP tool catalogs to query-relevant subsets.

See `docs/superpowers/specs/2026-05-19-rl-mcp-tool-gate-design.md` (in parent repo) for design.

## Setup
```bash
python -m venv .venv
source .venv/bin/activate    # or .venv\Scripts\activate on Windows
pip install -e ".[dev]"
```
